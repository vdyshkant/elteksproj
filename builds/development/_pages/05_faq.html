  • [ACS] Как настроить Dialplan на ONT при помощи ACS сервера?

    Для использования регулярных выражений в dialplan на ONT со стороны ACS-сервера необходимо:

    • ♦ выключить Outbound Proxy;
    • ♦ прописать Proxy Server и Registrar Server;
    • ♦ установить S-timer (время, в течении которого ONT будет ожидать продолжения набора, если уже набранный номер совпадает с каким-либо образцом в плане нумерации, но есть возможность получения большего количества цифр, что приведет к совпадению с другим образцом. По умолчанию он установлен в 4 сек.);
    • ♦ прописать DigitMap.

    Допустимые символы в DigitMap:

    • ♦ | - логическое ИЛИ - используется для разделения правил.
    • ♦ X или х – любая цифра от 0 до 9, равнозначно диапазону [0-9];
    • ♦ 0 - 9 – цифры от 0 до 9;
    • ♦ * – символ *;
    • ♦ – символ #;
    • ♦ Т - спецсимвол, который включает S-timer.
    • ♦ . – спецсимвол «точка» указывает на возможность повторения предшествующей перед данным символом цифры от нуля до бесконечности раз.
    • ♦ [ ] – указание диапазона (через тире), либо перечисление (без пробелов, запятых и прочих символов между цифрами), например:
    • ♦ диапазон [1-5] - 1,2,3,4 или 5;
    • ♦ перечисление [138] - 1,3 или 8;
    • ♦ диапазон и перечисление [0-9*#] – от 0 до 9, а также * и #.

    Например

    • ♦ set property “InternetGatewayDevice.Services.VoiceService.1.VoiceProfile.1.DigitMap” “(4xxx|2xT|2xxxxxx|8x.)”
    • ♦ set property “InternetGatewayDevice.Services.VoiceService.1.VoiceProfile.1.SIP.OutboundProxy” “0.0.0.0”
    • ♦ set property “InternetGatewayDevice.Services.VoiceService.1.VoiceProfile.1.SIP.ProxyServer” “192.168.18.95”
    • ♦ set property “InternetGatewayDevice.Services.VoiceService.1.VoiceProfile.1.SIP.RegistrarServer” “192.168.18.95”
    • ♦ set property “InternetGatewayDevice.Services.VoiceService.1.VoiceProfile.1.X_BROADCOM_COM_CriticalDigitTimer” “8000”
